# Dependências

### Back-End
yarn, express, mongoose, multer, sharp, cors, socket.io

#### Comandos 
yarn  init -y

yarn add express

yarn add nodemon -d

yarn add mongoose

yarn add multer

yarn add sharp

yarn cors

yarn socket.io

#### Como instalar as Depêndencias 
Basta digitar **yarn install no terminal**
